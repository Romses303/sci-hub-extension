# sci-hub-extension
Браузерное расширение для быстрого доступа к Sci-Hub через контекстное меню с возможностью выбора зеркала

Расширение для быстрого доступа к Sci-Hub через контекстное меню браузера.  
**Возможности**:  
- Поиск статей по выделенному тексту или ссылке.  
- Кастомизация списка зеркал Sci-Hub.  
- Автоматическая вставка запроса в поле ввода.  

## Установка
1. Скачайте репозиторий.  
2. В Chrome/Edge перейдите в `chrome://extensions`.  
3. Включите **«Режим разработчика»** и нажмите **«Загрузить распакованное расширение»**.  
4. Выберите папку с проектом.  

## Технологии
- JavaScript (Chrome Extensions API)  
- Manifest V3  
- HTML/CSS
